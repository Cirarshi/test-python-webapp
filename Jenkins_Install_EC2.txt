## Steps to Install Jenkins On EC2 machine
=================================================================================================================================

Req: Deploy a EC2 server with the security policy updated Custom TCP for port 8080 as 0.0.0.0/0 (Allow evberyone for port 8080)

====================================================================================================================================

ğŸ”¹ Step 1: Update Your VM:
```bash
sudo apt update
```

ğŸ”¹ Step 2: Install Java
Jenkins runs on Java (Java 17 or 21 recommended). Install it with:
```bash
sudo apt install openjdk-17-jdk -y
```
Verify:
```bash
java -version
```

ğŸ”¹ Step 3: Add Jenkins Repository

Add Jenkins key:
```bash
curl -fsSL https://pkg.jenkins.io/debian/jenkins.io-2023.key | sudo tee \
  /usr/share/keyrings/jenkins-keyring.asc > /dev/null
```

Add repo:
```bash
echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
  https://pkg.jenkins.io/debian binary/ | sudo tee \
  /etc/apt/sources.list.d/jenkins.list > /dev/null
```

ğŸ”¹ Step 4: Install Jenkins
```bash
sudo apt update
sudo apt install jenkins -y
```

ğŸ”¹ Step 5: Start and Enable Jenkins
```bash
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

Check status:
```bash
systemctl status jenkins
```

ğŸ”¹ Step 6: Access Jenkins
Open your browser and go to:
http://<your-ec2-instance-ip>:8080
Example: http://15.223.200.49:8080/
